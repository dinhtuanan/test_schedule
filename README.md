# Test Schedule python library
1. Python version: 3.9.13
1. Virtual Env: venv
## Create venv project
```
python -m venv schedule_env
```
## Activate venv
```
schedule_env\Scripts\activate
```

## Install python library
```
pip install -r requirements
```

## Run schedule task
```
python schedule_task.py
```
